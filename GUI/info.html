World Time

This app helps you get the time of any country in the world. not all country are available, but you can access all time zone.

World Time indicates the standard time zones. In theory time zones are based on the division of the world into twenty four time zones of 15 degrees longitude each. Read More
Features

Get the current time of any country in the world by time zone.

Installation
  1. Clone the repository.
  2. Install the required packages using pipinstall-r requirements.txt
Usage
  1. Run pythonworld_time.py
  2. Enter the name of the country or the time zone you want to get the time for.
